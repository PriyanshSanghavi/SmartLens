# SmartLens
This Project Smart Lens is an android Application developed for help to Traffic Police and Drivers. Using this application Police submit their attendance with face detection and location and also check drivers Idâ€™s and Document using face detection facility. Drivers can use this application for their access in various cities or areas without need to carry any Id and document. Police can give mamo online and driver can also pay fine online. Police can also ask for leave application and branches and admin respond it using this application and also check their attendance charts and given memo reports and also send them important circular easily.
